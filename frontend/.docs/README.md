## 🛠 Tech Stack

- **Framework**: Next.js 16.2.7 with TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui v4.10.0
- **Animations**: Aceternity UI v0.2.2 + Framer Motion v12.40.0
- **Build Tool**: ESLint for code quality

## 📋 Key Guidelines

### Before Writing Code
1. ✅ Read the guidelines
2. ✅ Check for existing components
3. ✅ Use TypeScript for all components
4. ✅ Follow the folder structure

### While Developing
1. ✅ Use Tailwind CSS for styling
2. ✅ Use shadcn/ui components
3. ✅ Support dark mode
4. ✅ Ensure responsive design (mobile-first)
5. ✅ Write accessible code

### Before Committing
1. ✅ Run `npm run lint`
2. ✅ Test in browser (light and dark mode)
3. ✅ Verify responsive design
4. ✅ Check accessibility
